    盒模型

    注意：IE6的content指的是除了margin之外的都是

  • 盒子默认有一个1px的边框，但是需要 border-style: dotted;把它显示出来
  • 盒子的宽度默认和父元素一致，高度根据内容撑开
  • 当为父元素设置常和宽时，子元素的padding设置之后，就会将父元素撑大； 当父元素的常和宽没有具体设定时，子元素的padding设置没有将父元素撑大
  • margin:0 auto;设置居中
  • 两个盒子同时设置左右边距的时候，他们的距离是margin之和；两个盒子同时设置上下边距的时候 ，他们的距离是较大的那个margin

  • 边框样式border-style:dashed; 边框宽度border-width:2px; 边框颜色border-color:red; 合体border:1px solid green;//ridge垄状边框

    border-style:solid double dashed ridge 同时指定四个方向的样式，width和color类同 如果指定四个值：上 右 下 左 如果指定三个值：上 左右 下 如果指定两个值：上下 左右 如果指定一个值：上下左右全部使用该值

    浮动

    内联元素设置为float之后就变成了块级元素

    清除浮动的方式
  • 给浮动的父元素也加上float:left(这种方式大多数情况下不建议用)
  • 在浮动的父元素加上overflow：hidden；
  • ::after{ content："";display:block;clear:both;}
  • 在浮动元素的后面加上 < div style="clear:both">.< /div >
  • clear:both;
  • /以下这种方式兼容性最好/ 如果.clearfix是浮动的父元素，那么 .clearfix:after{content:"\200B";display:block;height:0;clear:both;} .clearfix{*zoom:1;}/*IE/7/6*/

    • 内联元素display：inline 块级元素display：block 内联-块级元素display：inline-block

    定位

  • position:relative相对定位指的是相对于元素本身的位置，不脱离文本流，不改变元素的性质，使元素上升一层
  • position:absolute绝对定位指的是相对于设置了position属性最近的祖先元素，如果没有，就相对于浏览器窗口(body)。绝对定位会脱离文本流，元素设置这个属性后就变成了块元素
  • position:fixed固定定位相对于浏览器窗口，但是这个属性在ie6中不支持
  • position属性设置后，即可使用top,right,bottom,left和z-index属性，否则这些属性无效

    • 居中

  • 行内元素居中text-align:center,vertical-align: middle
  • 块级元素居中margin:0 auto;line-height:该元素的行高
  • 居中一个浮动元素： 
    
    .div {
    width:500px ; height:300px;//高度可以不设
    margin: -150px 0 0 -250px;
    position:relative;         //相对定位
    background-color:pink;     //方便看效果
    left:50%;
    top:50%;
    }
  • 让绝对定位的div居中

    width: 1200px;

    margin: 0 auto;
    top: 0;

    图片整合(雪碧图)

  • 好处：节省图片的空间大小，提高浏览器加载图片的速度
  • 使用步骤：1.设置一个大的宽和高，background-size设置为200px，background-position随便设2.在浏览器先对 position进行微调到左上角3.掉宽高调整到合适的大小

    • Photoshop

      操作快捷键

    1. 用四条网格线拖成一个矩形框之后，用虚线框选中，按F8即可查看矩形的宽和高
    2. 要选择psd文件中的文字，需要先点击ps左边的T选择项。字体的颜色大小样式都在正上方的菜单栏中
    3. 测量前注意要在右边导航栏右键>首选项>单位和标尺中设置单位为像素
    4. 裁剪图片时，需要将其选中，然后点击图像>裁剪>存储为web和设备所用格式>点击存储。点击右边的历史记录可以返回到原来的状态
    5. 裁图的时候，如果图片有多个图层，需要点击左边的鼠标菜单>选择自动选择>选中图层>点一下图片，然后在右边的图层菜单中，去掉不想要的背景图层
    6. 制作雪碧图（作为背景图片使用）的步骤：用ps打开某一个png图片，选择画布，填写宽高，选择定位来调整方向，然后将别的图片裁剪到当前画布，调整一下位置>存储为web格式png图片
    7. 在另存为web图片的时候，如果图片不是透明背景的，就应该选用空间大小最少的格式保存（通常gif格式比较小）
    8. 选择框选择区域时，如果大一点或者小一点，需要点击左侧选择框，选择上方的修补菜单栏进行填补或裁撤图片
    9. 常用快捷键: 全屏tab键 画布调整到100%Ctrl+1 调整到适应窗口大小Ctrl+0 在选择图片的时候点左侧尖角菜单>点击相应图片>右侧图层栏中按ctrl去点相应图层就选中图片>按Alt把左边小眼睛 去掉，背景图层就出来了，然后再点裁剪 如果想直接裁切出多个图层的时候点击图片，在右侧图层中选择多项图层 按Alt现在几乎所有的图层都消失了，再次点击小眼睛选中需要的图层，点击上侧裁切菜单>选择透明像素>点击确定>另存为web... 如果想精准测量宽和高，就选中右侧所有图层，然后拖网格线，在图边缘松开，网格线就会吸附到准确的位置 Ctrl+H 可以隐藏网格线
    10. 切图时要选择左侧的切图工具，给所要选择的图片加上网格线，用切图工具一个一个划出范围，另存为web格式...
    11. 编辑-自由变换（ctrl+t）怎样图片去水印 仿制图章工具（alt键选取）怎样填充颜色选区-shift+backspace 选择上方菜单栏可以对图片进行液化操作
    12. 钢笔工具钢笔工具快捷键：ctrl,alt(分别用来操作锚点和手柄)可以抠图和画logo
    13. 动画制作自制动画： 窗口-动画 复制帧，删除帧 播放动画，步进 移动位置动画，透明度动画 循环次数 过渡动画帧 动画调板 另外可以导入视频制作动画文件>导入视频帧>导入之后按shift按进行帧区间的选取>另存为web格式..

    样式初始化

      原因
    • - 因为浏览器的兼容问题，不同浏览器对有些标签的默认值是不同的，如果没对CSS初始化往往会出现浏览器之间的页面显示差异。
    • - 当然，初始化样式会对SEO有一定的影响，但鱼和熊掌不可兼得，但力求影响最小的情况下初始化。
