• 基线的理解
  • 元素隐藏：visibility:hidden;（不可见，占位）display:none（不可见，不占位）
  • 透明度：opacity会使元素本身和其子元素都透明；rgba只是作用于背景色
  • windows默认的字体在：c/windows/fonts路径下。 chrome设置字体，工具栏-->设置-->高级设置-->字体。 地址栏地址解析：C:/Windows/System32/drivers/etc/目录下的hosts文件，把localhost修改成其他名字
  • @media选择器 @media(max-width:400px){...} :文档宽度小于400px时（最大为400px时）;@media(min-width:300px){...} :文档宽度大于300px时（最小为300px时）; 以上主要应用于pc端。 @media(max-device-width:640px){}用于移动端设备
  • IE5,6使用的是quirk模式，行内元素也可以设置宽高.所以要避免就要设置doctype
  • border-radius:10px出现圆角效果，border-radius:50%;就可以实现圆形效果，border-radius:100px(水平方向半径)/50px(垂直方向半径):做一个椭圆
  • cursor可以改变鼠标的形状，如：cursor:pointer手型
  • 制作三角形 
    
        div{
            width:0;
            height:0;
            border-top:30px solid transparent;
            border-left:30px solid transparent;
            border-right:30px solid transparent;
            border-bottom:30px solid green;
        }
  • 跳转到相应的位置： <a href="#p1">目标</a> <p id="p1">目标段落</p>
  • p:target{border:1px dotted red} 承接上一个内容，表示点击之后目标段落加上红色虚线效果
  • :first-child :last-child :nth-child结构化伪类（第一个，最后一个,第n个）如：:nth-child(3n) :nth-child(odd)
  • 制作透明图片，打开ppt>给图片设置画布>菜单栏选中取色器>取图片中白色的部分>ok
  • 居中的两种思路：父元素relative>目标居中元素absolute，top:50%，left:50%,margin-left:目标的宽度/2,margin-top:目标的高度/2>ok; 父元素display:table-cell,text-align:center,vertical-align:middle>ok
  • 其他居中：两个内联元素放一起，span+input如果input元素并没有和span一条水平线上，可以vertical-align:middle
  • display:table-cell 其父元素display:table css3新增的单元格布局,可以像普通的单元格那样布局，主要用来做响应式布局
  • 计算函数width: calc(100%-20px)
    • 二级菜单纯css制作(类似淘宝京东、图片大小自适应) 京东移动端界面 聚美优品（刘重） 聚美优品（触屏版） 聚美优品（其他）

    动画

  • transition:过渡 例如：.box{background:red} .box:hover{background:green;transition:all 2s linear(这个是速率，可以用cubic-bezier(0,1.5,1,-0.38)代替)}
  • transform:缩放、移动、倾斜、旋转 例如：transform:rotate(45deg)--transform:scale(2)--transform:translate(20px,20px)--transform:skew(45deg,45deg)
  • animation:动画
    • 幽灵按钮 会飞的鸟 跳跳球 瀑布流 手机手势滑动
