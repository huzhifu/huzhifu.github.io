基本知识

  • window.$===window.jQuery===jQuery
  • 两把利器，jQuery核心函数(如：$('p'))，jQuery对象（执行jQuery函数返回的对象,如$('p').val()），不管执行其任何方法都将自身返回
  • $(ele)将dom对象封装为jQuery对象，$('selector')[0],:将jQuery对象转化为dom对象
  • jQuery基本行为：each() size()/length [index]/get() index()
  • 层次选择器，用来查找某个元素的孩子，兄弟...(亲属)如：$('div>.content').css('background','red')【也可以传对象，里面是样式名值对】;过滤选择器：在原有选择器匹配的基础上进一步过滤，如$('div:first')
  • 得到页面滚动条的scrollTop $(document.body).scrollTop()+$(document.documentElement).scrollTop() 设置滚动条坐标：$('body,html').scrollTop(60)
  • jQuery文档筛选模块，过滤是从当前满足条件的一堆元素找到某一个或某一类；查找是从当前元素找到其亲属元素（父子、兄弟节点...）
    • * 属性/HTML/val
      • * 操作任意属性
      attr(): 读取某个属性值, 设置属性
      removeAttr(name)
      prop(): prop('checked', true/false);
      * 操作class属性
      addClass() : 添加class属性值, 原来的值还在
      removeClass(): 移除属性值
      * 操作HTML代码/文本/值
      html() : 标签体文本
      //text()
      val() : value属性值
    • *CSS
      • 设置css样式/读取css值
      css() : 元素的样式属性(可读写) 操作的是内联样式
      获取/设置标签的位置数据
      offset() : 相对页面左上角的坐标(可读写) : 包含坐标值的对象 {left: xxx, top:yyy}
      position() : 相对于父元素左上角的坐标 (只读) : 包含坐标值的对象 {left: xxx, top:yyy}
      scrollTop() : scrollLeft() : 滚动条坐标 (可读写)
      获取/设置标签的尺寸数据
      heigh() / width() : 不包括内边距和边框的尺寸
      innerHeight() / innerWidth() : 包括内边距,但不包括边框的尺寸
      outerHeight() / outerWidth() : 包括内边距和边框的尺寸(参数为true还包括外边距)
    • *文档
      • 添加节点
      append(content) : 向当前匹配的所有元素内部的最后插入指定内容
      appendTo(selector): 将当前标签插入到selector对应的标签内部的最后
      prepend(content) : 向当前匹配的所有元素内部的最前面插入指定内容
      before(content) : 将指定内容插入到当前所有匹配元素的前面
      after(content) : 将指定内容插入到当前所有匹配元素的后面
      替换节点
      replaceWith(content) : 用指定内容替换所有匹配的标签
      删除节点
      empty() : 删除所有匹配元素的子元素
      remove() : 删除所有匹配的元素
    • *筛选
      • 过滤 : 在jQuery对象中的元素对象数组中过滤出一部分元素来
      first()
      last()
      eq(index|-index)
      filter(selector) : 在当前元素中找
      not(selector)
      has(selector)
      查找: 在已经匹配出的元素集合中根据选择器查找孩子/父母/兄弟标签
      children() : 找孩子
      find() : 找后代
      parent()
      prevall()
      nextall()
      siblings()
    • *事件
      • 事件处理
      on(事件名, funn) : 通用的绑定事件监听
      xxx(fun) : 绑定某个具体的事件监听
      off(事件名) : 解绑事件监听
      事件切换
      hover(fun1, fun2) : 同时绑定鼠标移入和移出监听
      事件委派(委托)
      delegate(selector, 事件名, fun) : 绑定委托事件监听
      undelegate() : 移除委托事件监听
      相关重要知识
      事件坐标
      event.clientX | event.clientY : 相对可视窗口左上角
      event.pageX | event.pageY : 相对页面左上角
      event.offsetX | event.offsetY : 以offsetParent(已定位的父辈元素)的左上角为原点
      阻止事件冒泡
      event.stopPropagation();
      取消事件的默认行为
      event.preventDefault();
      return false;
    • *效果
      • 滑动动画
      原理: 不断改变元素的高度来实现的
      slideDown() : 带动画的展开
      slideUp() : 带动画的收缩
      slideToggle() : 带动画的切换展开/收缩
      淡入淡出动画
      原理: 不断改变元素的透明度来实现的
      fadeIn() : 带动画的显示
      fadeOut() : 带动画隐藏
      fadeToggle() : 带动画切换显示/隐藏
      显示/隐藏动画
      原理: 不断改变元素的尺寸和透明度来实现
      show() : (不)带动画的显示 display='block/inline'
      hide() : (不)带动画的隐藏
      toggle() : (不)带动画的切换显示/隐藏
      自定义动画
      animate() : 自定义动画效果的动画
      stop() : 停止动画
    • * jQuery对象
      • 自定义动画
      animate({}, speed, 'linear', fn) : 自定义动画效果的动画
      stop() : 停止动画
    • * 插件机制
      • jQuery.fn.extend(object)
      扩展jQuery对象的方法, object中的方法会成为jQuery对象的方法
      注意方法中this的问题
      jQuery.extend(object)
      扩展jQuery核心函数的方法, object中的方法会成为$的方法
    • * 多库共存
      • jQuery.noConflict() : 执行完此方法后, jQuery不再使用$, 这样其它框架就可以使用$了
